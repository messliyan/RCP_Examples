1.6.3  视图操作

1.6.3.1 查看数据

①、右键视图节点，出现查看数据菜单，如下图

 

②、单击查看数据，出现查看数据面板及其数据信息如下图

1.6.3.2 复制视图名

①、右键视图节点，出现复制菜单如图

②、单击复制，视图名自动复制到系统粘贴板上

1.6.3.3 删除视图

①、右键视图节点，出现删除菜单如图

②、单击删除， 出现删除提示框

③、单击查看sql,出现删除视图sql面板如下图

单击复制，删除视图语句自动复制到系统粘贴板上

④、单击确定，删除视图，原视图在原节点上删除

 

 

1.6.3.4 刷新视图

①、右键视图节点出现刷新菜单如图：

②、右键单击刷新，进行视图的刷新
