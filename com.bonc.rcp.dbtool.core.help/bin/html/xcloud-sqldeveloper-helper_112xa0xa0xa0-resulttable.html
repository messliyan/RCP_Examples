1.12    查询结果表展示

1.12.1  调整行高，单行或多行高亮选中

调整行高，单行或多行高亮选中

①、调整行高：查询结果表鼠标放在行号处调整单行行高，或者在行号处使用shift键、ctrl键选中多行来调整选中行行高(行高取决于选中行最后一行拖拽的宽度)。

②、在行号处(或选中多行)通过右键选择 自动调整行尺寸 调整行高（自适应）
