①、右键单击数据库实例节点，出现删除菜单。如图

②、点击删除弹出如图所示的对话框。

 

③、点击查看SQL，出现如图所示的对话框，可以查看删除数据库的SQL语句，单击复制可以对SQL语句进行复制。
